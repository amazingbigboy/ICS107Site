This navigation has drop down menu. The first button has no drop down list, but the next two buttons have drop down lists. Hover over the menu item to drop the list down. Unlike javascript drop-down lists, this meets accessibility standards. The list can be displayed without any CSS applied and all menu choices will be displayed.

This is an efficient way of organizing horizontal navigation. The audience isn't overwhelmed with so many initial choices and all the pages can be listed. The width of the items might need to be changed depending upon how many characters are it the line item.

One caveat concerning these drop down list is if the content beneath the navigation isn't long enough and the navigation is at the bottom of the page, not all the navigation items can be seen, so that is a usability concern. Another usability concern is that the top item in the menu are not click-able, they just lead to the drop down list. One might create pages for those two items then have the items on the drop list on these pages.
