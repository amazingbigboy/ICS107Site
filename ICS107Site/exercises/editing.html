No More Tables

Before there was CSS, pages were laid out with tables. Some sites still have this legacy coding which was used to create their sites. It can be difficult to make changes to the site, because the structure is so rigid. It's important to convert site from layout with tables to layout with CSS.

 

 

For instance if you had numerous pages with this layout and you wanted to change the layout, it would be a major project often requiring recoding each and every page.

To move out of a table layout,
If the menu is in a table layout, convert the table cells to line items and the row to unordered list.
Convert the main table tags to wholepage division.
Convert rows into divisions
Convert cells into divisions
Add in CSS to match the table layout or improve on it.
